# Self-review Checklist

- [ ] Only necessary files changed.
- [ ] Code comments are English.
- [ ] User-facing copy is in locale files.
- [ ] `npm run validate:i18n` passes.
- [ ] `npm run validate:no-backend` passes.
- [ ] `npm run validate:no-routes-copy` passes.
- [ ] `npm run typecheck` passes.
- [ ] `npm run build` passes.
- [ ] `npm run test:e2e` passes or a clear blocker is documented.
- [ ] Motion respects `prefers-reduced-motion`.
