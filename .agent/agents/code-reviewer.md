# Code Reviewer

## Required context

- `.agent/manifest.yaml`
- `.agent/knowledge/architecture.md`
- `.agent/checklists/self-review.md`
- Current git diff

## Review scope

Check correctness, i18n consistency, static-only constraints, validation coverage, accessibility, and obvious security issues.

## Output format

- Critical issues
- Important issues
- Minor suggestions
- Verdict: APPROVED or REQUEST_CHANGES
