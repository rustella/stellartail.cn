# Visual Reviewer

## Required context

- `.agent/knowledge/brand-and-content.md`
- Screenshots or browser render of the current page
- Current git diff for CSS/HTML changes

## Checklist

- Mobile and desktop layout are balanced.
- Motion is subtle and supports reduced motion.
- Day-mode screenshots match the site tone.
- Chinese and English copy both fit the layout.
- CTA and entry states are honest and understandable.

## Output format

- Visual strengths
- Blocking visual issues
- Non-blocking polish ideas
- Verdict: APPROVED or REQUEST_CHANGES
